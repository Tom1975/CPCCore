Amstrad CPC cross platform core emulation.
This can be used as a library for any project.

I use it for :

- SugarConvDsk
- SugarConvTape
- SugarLibRetro
- Sugarbox

It contains user tests (mainly to check no regression occurs, especially on protection and FDC emulation)
